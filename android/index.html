Google Developers Study Jams é uma série livre global, gerenciado pela comunidade, sobre grupos de estudo sobre determinado assunto. A série atual de estudos Jams é apresentada em parceria com Udacity e seguirá o com o curso de Android para Iniciantes (Android for Beginners).

Study Jams são uma oportunidade para que os alunos recebem, orientação pessoal e tutoria livre no curso Udacity em um ambiente de sala de aula divertida.

Promova usando a hashtag #devstudyjam

Como fazer um App Android

Este curso é destinado aos estudantes que são novos na programação, e querem aprender como construir aplicativos Android. Você não precisa ter nenhuma experiência de programação para fazer este curso. Se você estiver usando um smartphone para navegar na web e bate-papo com os amigos, então você é o nosso alvo perfeito!

Como fazer um App Android

Aprender alguma coisa nova pode ser difícil. Vamos orientá-lo através do processo de fazer aplicações Android, mas para tirar o máximo proveito deste curso, você deve trazer o seu entusiasmo para aprender, e reservar um tempo em sua agenda para aprender com a gente.

Como fazer um App Android
